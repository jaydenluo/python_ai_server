# 🏗️ 架构设计

## 📚 架构文档

### 1. [架构概览](overview.md)
**内容**: 框架整体架构设计
- 分层架构
- 模块划分
- 设计原则
- 技术栈

### 2. [路由系统](routing.md)
**内容**: 注解路由系统
- 路由装饰器

- 权限验证
- 数据权限
- 字段权限

### 5. [服务层架构](service-layer.md)

**内容**: 智能 ORM 系统

- 迁移系统
